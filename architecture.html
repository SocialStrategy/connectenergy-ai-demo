Technical Spec

Autonomous Commercial Solar Pipeline

End-to-end technical architecture specification for the ConnectEnergy automated AI outbound sales engine.

0. Core Orchestration

Framework

OpenClaw Agent Framework running autonomous background CRON jobs. Adheres to the n8n principle: discrete, isolated scripts passing state via a shared database to prevent LLM hallucination during multi-step tasks.

State & Data

SQLite (WAL mode) for transactional pipeline tracking. Cloud object storage (AWS S3 or Cloudflare Blob) for satellite imagery, generated renders, and PDF artifacts.

1

Target Prospecting & Intelligence

Identifying viable commercial targets and finding the correct decision-makers.

  • API
    Google Places API / Cadastral Data Scans specific industrial zones for logistics, manufacturing, and warehouse footprints. Extracts business names and addresses.
  • API
    Apollo.io / Hunter.io Enriches the company data to find the specific "Facility Manager" or "VP of Operations" and verifies their email address.
2

Satellite Extraction & Roof Analysis

Validating roof viability and determining usable square meterage.

  • DATA
    Nearmap API / Google Maps Aerial Pulls recent, high-resolution top-down and oblique satellite imagery of the specific property coordinates.
  • AI
    Gemini 3.1 Pro Vision Analyzes the image to detect existing solar (discards lead if present), identifies HVAC obstructions/skylights, and estimates the contiguous usable roof space.
3

System Sizing & Financial Modeling

Calculating the optimal solar array size, energy yield, and ROI.

  • API
    NREL PVWatts / OpenSolar API Takes the coordinates and usable roof area to calculate optimal kW capacity and estimated annual kWh yield.
  • CODE
    Financial Logic Engine Deterministic Python/Node script calculating STC rebates and payback periods based on localized commercial tariffs (e.g., Energex QLD rates).
4

Generative Visual Rendering

Creating the visual "hook" for the outbound email.

  • AI
    Stable Diffusion / Flux.1 (via Replicate API) Uses an image-to-image pipeline with a custom LoRA and ControlNet masking. The mask ensures the photorealistic flush-mount solar panels only render on the empty roof space, matching the sun's exact lighting and shadow angles from the original satellite image.
5

Proposal Generation & Dispatch

Generating the final artifact and cueing it up for a human to review and send.

  • CODE
    Playwright PDF Generator Injects the calculations, ROI data, and the generative "After" image into a ConnectEnergy branded HTML template, printing it to a polished PDF.
  • API
    Pipereply CRM Integration Hits the Pipereply webhook/API to create a new Lead. Attaches the PDF and drafts a hyper-personalized email containing the key metrics. The email sits in "Drafts" for Sam Watson to click approve/send, maintaining safety and quality control.
